
# Network Scanner Tool (Python)

## Overview
This project is a Python-based Network Scanner Tool designed to discover active hosts
and open ports within a given network range. It demonstrates basic network reconnaissance
techniques used in cybersecurity.

## Features
- IP range scanning
- Host availability detection (ping-based)
- TCP port scanning
- Fast execution using threading
- Clear command-line interface

## Technologies Used
- Python 3
- socket
- threading
- subprocess

## Usage
```bash
python main.py 192.168.1.1 192.168.1.254
```

## Disclaimer
Scan only networks you own or have permission to test.
